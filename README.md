# Projeto-Integrador--SSNE
Projeto Integrador - Sistema de Suporte para Novos Estudantes.

O SSNE tem o objetivo de facilitar a adaptação dos novos estudantes do IFRN por meio de uma plataforma web focada em um mapa digital interativo para localização fácil de salas e setores. Resolvemos criá-la para centralizar informações e o mapa em um só lugar, reduzindo a desorientação dos calouros nas primeiras semanas e diminuindo a sobrecarga de atendimentos repetitivos nas secretarias e coordenações do campus.
